This is mostly just here as an easy way to have a regularly-updated reference I can link specific lines for. Contributing is not a primary consideration, but is not discouraged.

The process for contributing is not going to be like most repositories. Open a pull request if you like, but if it ends up incorporated into the base raws, it'll be via applying the patch on our ends locally, so the PR won't be *merged*, it'll just show up in the next commit. I'll close it with an "incorporated in DF xx.xx" if this happens.
